# DMart-Analysis_Dashboard
Power BI project for DMart Sales Analysis

## Project Overview
This project analyzes D-Mart sales data using Power BI to understand sales performance, customer behavior, and product trends. 
The dashboard provides interactive visual insights to support business decision-making.

## Objective
- Analyze overall sales and profit performance
- Identify top-performing products and categories
- Understand customer purchasing patterns
- Track regional and time-based sales trends

## Dataset Description
The dataset includes the following fields:
- Order ID
- Order Date
- Customer Name
- Region / City
- Product Category
- Product Name
- Quantity
- Sales
- Profit
- Payment Mode

## Key Metrics (KPIs)
- Total Sales
- Total Profit
- Total Quantity Sold
- Average Order Value
- Profit Margin

## Dashboard Features
- Sales and profit trends over time
- Category-wise and product-wise sales analysis
- Region-wise sales performance
- Top-selling and low-performing products
- Interactive filters (date, region, category)

## Tools & Technologies
- Power BI
- Excel / CSV dataset
- DAX for calculated measures

## Insights Generated
- Identified high-revenue product categories
- Observed seasonal sales patterns
- Highlighted regions with highest and lowest profitability
- Detected products with high sales but low profit margins

## Business Use Cases
- Helps management optimize inventory planning
- Supports pricing and discount strategies
- Assists in identifying growth opportunities across regions
